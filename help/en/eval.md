- `eval (TRY)`
- `eval (TRY, CATCH)`

Parse the string `TRY` and evaluate it as if it were program text.

If `CATCH` is supplied and evaluation of `TRY` fails, evaluate `CATCH`.

See also: `evalin`, `assignin`, `feval`.
