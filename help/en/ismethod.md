- `TF = ismethod (OBJ, "METHOD")`
- `TF = ismethod ("CLASSNAME", "METHOD")`

Return true if `METHOD` is a method of the object `OBJ` or of the named class
`CLASSNAME`.

See also: `methods`, `isprop`, `isobject`.
