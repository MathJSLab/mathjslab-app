- `TF = isclass (NAME)`
- `TF = isclass (OBJ)`

Return true if `NAME` names a class, or if `OBJ` represents a class definition.

This function is useful when checking class definitions before using class
introspection functions.

See also: `class`, `isa`, `metaclass`.
