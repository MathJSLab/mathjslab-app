- `TF = iscolumn (X)`

Return `true` if `X` is a column vector.

A column vector is a 2-D array for which `size (X)` returns `%[N, 1]%` with
non-negative `%N%`.

See also: `isrow`, `isscalar`, `isvector`, `ismatrix`, `size`.
