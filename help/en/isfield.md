- `TF = isfield (S, "NAME")`
- `TF = isfield (S, NAME)`

Return true if `S` is a structure and includes a field named `NAME`.

If `NAME` is a cell array of strings, return a logical array of the same size.

See also: `fieldnames`, `struct`.
