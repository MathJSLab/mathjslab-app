- `which NAME`
- `which NAME1 NAME2 ...`
- `S = which ("NAME")`

Display information about the symbol named `NAME`.

The result identifies whether the name resolves to a variable, built-in
function, function file, class, or other callable object.

See also: `exist`, `help`, `mfilename`.
