- `CLASSNAME = class (OBJ)`
- `OBJ = class (S, ID)`
- `OBJ = class (S, ID, PARENT, ...)`

Return the class name of `OBJ`, or create an object from the structure `S` with
class name `ID`.

Additional arguments name parent classes from which the new class is derived.

See also: `isa`, `typeinfo`.
