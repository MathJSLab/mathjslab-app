- `run SCRIPT`
- `run ("SCRIPT")`

Ejecuta `SCRIPT` en el espacio de trabajo actual.

Si el nombre del script no incluye una ruta, se busca en la ruta de carga
actual.

Véase también: `source`, `eval`.
