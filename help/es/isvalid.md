- `TF = isvalid (OBJ)`

Devuelve verdadero para objetos handle válidos y falso para objetos handle
eliminados o inválidos de otra forma.

Para matrices de objetos, devuelve una matriz lógica con un resultado para cada
objeto.

Véase también: `isobject`, `delete`.
