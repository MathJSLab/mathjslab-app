- `TF = isreal (X)`

Devuelve verdadero si `X` no es complejo.

Para matrices numéricas, el resultado es falso cuando cualquier elemento tiene
parte imaginaria distinta de cero.

Véase también: `real`, `imag`, `isnumeric`.
