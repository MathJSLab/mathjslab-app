* `~`

Operador lógico **NOT**.

El símbolo también se puede utilizar para descartar salidas de una
función que no sean deseadas sin utilizar una variable temporal.

>> `[~, IDX_OF_MAX] = max (X)`

Véase también: `!`, `not`.
