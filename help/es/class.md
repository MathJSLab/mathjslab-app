- `CLASSNAME = class (OBJ)`
- `OBJ = class (S, ID)`
- `OBJ = class (S, ID, PARENT, ...)`

Devuelve el nombre de la clase de `OBJ`, o crea un objeto a partir de la
estructura `S` con nombre de clase `ID`.

Los argumentos adicionales nombran clases padre de las cuales deriva la nueva
clase.

Véase también: `isa`, `typeinfo`.
