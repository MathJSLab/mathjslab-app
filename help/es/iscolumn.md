- `TF = iscolumn (X)`

Devuelve `true` se `X` es un vector de columna.

Un vector de columna es una matriz 2-D para la cual `size(X)` devuelve %%[N,
1]`% con %`N%% no negativo.

Véase tambiém: `isrow`, `isscalar`, `isvector`, `ismatrix`, `size`.
