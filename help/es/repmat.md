- `B = repmat (A, M, N)`
- `B = repmat (A, M, N, P, ...)`
- `B = repmat (A, [M N ...])`

Repite copias de `A` para formar una matriz mayor.

Los conteos de repetición pueden proporcionarse como argumentos escalares
separados o como un vector de dimensiones.

Véase también: `reshape`, `ones`, `kron`.
