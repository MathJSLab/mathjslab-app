- `which NAME`
- `which NAME1 NAME2 ...`
- `S = which ("NAME")`

Muestra información sobre el símbolo nombrado por `NAME`.

El resultado identifica si el nombre se resuelve como una variable, función
incorporada, archivo de función, clase u otro objeto invocable.

Véase también: `exist`, `help`, `mfilename`.
