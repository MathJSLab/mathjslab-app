* `Y = adj (X)`

Calcula la matriz adjunta para una matriz cuadrada `X`.

La matriz adjunta está definida por

>> %%adj (X) = cofactor (X)'%%.

Para la matriz invertible `X` es válida la siguiente relación:

%%%1/det(X)*adj(X) = inv(X)%%%

Véase también: `inv`, `cofactor`, `ctranspose`.
