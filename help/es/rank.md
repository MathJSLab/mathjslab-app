- `K = rank (A)`
- `K = rank (A, TOL)`

Calcula el rango numérico de la matriz `A`.

El rango se determina a partir de los valores singulares de `A`. Los valores
singulares menores que la tolerancia `TOL` se tratan como cero.

Véase también: `cond`, `svd`, `qr`.
