- `C = exist (NAME)`
- `C = exist (NAME, TYPE)`

Verifica si `NAME` existe como variable, función, archivo, directorio o clase.

El argumento opcional `TYPE` restringe el tipo de objeto que se busca.

Véase también: `which`, `iskeyword`.
