- `NAME = inputname (N)`
- `NAME = inputname (N, IDS_ONLY)`

Retorna o nome do N-ésimo argumento de entrada passado para a função chamadora.

Se o argumento de entrada não for uma variável nomeada, retorna uma string
vazia.

Veja também: `nargin`.
