- `TF = isvalid (OBJ)`

Retorna verdadeiro para objetos handle válidos e falso para objetos handle
excluídos ou inválidos de outra forma.

Para matrizes de objetos, retorna uma matriz lógica com um resultado para cada
objeto.

Veja também: `isobject`, `delete`.
