* `C = plus(A, B)`
* `C = plus (A1, A2, ...)`

This function and `A + B` are equivalent.

If more arguments are given, the summation is applied cumulatively
from left to right:

`(...((A1 + A2) + A3) + ...)`

See also: `minus`, `uplus`.
