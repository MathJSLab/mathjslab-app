- `TF = isclass (NAME)`
- `TF = isclass (OBJ)`

Retorna verdadeiro se `NAME` nomeia uma classe, ou se `OBJ` representa uma
definição de classe.

Esta função é útil ao verificar definições de classe antes de usar funções de
introspecção de classes.

Veja também: `class`, `isa`, `metaclass`.
