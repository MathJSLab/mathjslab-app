- `CLASSNAME = class (OBJ)`
- `OBJ = class (S, ID)`
- `OBJ = class (S, ID, PARENT, ...)`

Retorna o nome da classe de `OBJ`, ou cria um objeto a partir da estrutura `S`
com nome de classe `ID`.

Argumentos adicionais nomeiam classes pai das quais a nova classe é derivada.

Veja também: `isa`, `typeinfo`.
