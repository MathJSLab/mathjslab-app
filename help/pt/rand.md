- `X = rand (N)`
- `X = rand (M, N, ...)`
- `X = rand ([M N ...])`

Retorna uma matriz com valores aleatórios distribuídos uniformemente no
intervalo (0, 1).

Os argumentos são tratados da mesma forma que os argumentos para `eye`.

Veja também: `randn`, `rande`, `randg`, `randp`.
