- `TF = isreal (X)`

Retorna verdadeiro se `X` não é complexo.

Para matrizes numéricas, o resultado é falso quando qualquer elemento tem parte
imaginária diferente de zero.

Veja também: `real`, `imag`, `isnumeric`.
