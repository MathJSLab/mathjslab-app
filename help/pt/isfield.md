- `TF = isfield (S, "NAME")`
- `TF = isfield (S, NAME)`

Retorna verdadeiro se `S` é uma estrutura e inclui um campo chamado `NAME`.

Se `NAME` for uma matriz celular de strings, retorna uma matriz lógica do mesmo
tamanho.

Veja também: `fieldnames`, `struct`.
