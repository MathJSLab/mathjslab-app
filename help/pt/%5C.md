* `\`

Operador de divisão à esquerda.

Dentro de strings entre aspas duplas, `\` é o marcador de continuação de linha
usado para juntar a linha atual com a linha seguinte.

Veja também: `mldivide`, `.\`, `ldivide`, `/`.
