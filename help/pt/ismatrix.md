- `TF = ismatrix (X)`

Retorna `true` se `X` for uma matriz 2-D.

Uma matriz é um objeto com duas dimensões (`ndims (X) == 2`) para o qual
`size (X)` retorna %%[M, N]%% com %%M%% e %%N%% não negativos.

Veja também: `isscalar`, `isvector`, `iscell`, `isstruct`, `issparse`, `isa`.
