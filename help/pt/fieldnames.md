- `NAMES = fieldnames (S)`
- `NAMES = fieldnames (OBJ)`
- `NAMES = fieldnames (JAVAOBJ)`
- `NAMES = fieldnames ("JAVACLASSNAME")`

Retorna uma matriz celular de strings com os nomes dos campos da entrada
especificada.

Para uma estrutura, retorna os nomes dos campos da estrutura. Para um objeto,
retorna os nomes das propriedades públicas.

Veja também: `isfield`, `struct`, `methods`.
