- `TF = isprop (OBJ, "PROP")`

Retorna verdadeiro se `PROP` é uma propriedade do objeto `OBJ`.

Se `OBJ` for uma matriz de objetos, retorna uma matriz lógica indicando se cada
objeto possui a propriedade.

Veja também: `properties`, `ismethod`, `isobject`.
