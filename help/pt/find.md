- `IDX = find (X)`
- `IDX = find (X, N)`
- `IDX = find (X, N, DIRECTION)`
- `[I, J] = find (...)`
- `[I, J, V] = find (...)`

Retorna os índices dos elementos não nulos de `X`.

Com duas saídas, retorna os subscritos de linha e coluna. Com três saídas,
também retorna os valores não nulos. Os argumentos opcionais `N` e `DIRECTION`
limitam a busca às primeiras ou últimas correspondências.

Veja também: `nonzeros`.
