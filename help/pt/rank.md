- `K = rank (A)`
- `K = rank (A, TOL)`

Calcula o posto numérico da matriz `A`.

O posto é determinado a partir dos valores singulares de `A`. Valores
singulares menores que a tolerância `TOL` são tratados como zero.

Veja também: `cond`, `svd`, `qr`.
